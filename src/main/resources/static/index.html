ScholarWiki

Lightweight vector-powered academic paper wiki engine. Store, retrieve, and semantically search research paper notes with ease.

All systems operational

About

📄

Paper Notes

Submit pre-processed paper reports with structured markdown. Automatic chunking and vectorization for semantic retrieval.

🔍

Vector Search

Semantic search across all paper chunks using embedding-based cosine similarity. Find relevant content by meaning, not keywords.

🏷️

Tag Navigation

Agent-assigned tags for browsing and filtering papers by topic. Tags evolve as the agent's classification improves.

📝

Annotations

Simple paper-level notes in GitHub Issues style. No complex anchoring — just thoughts attached to papers.

Flow

Local Agent → POST /papers → Chunk Parse → Vectorize → SeekDB
